## Enviando para o repositorio

### 1º Metodo

Inicia e termina cada etapa antes de iniciar a próxima.

Este método é usado quando é necessário que algo termine antes de iniciar outro processo. Um exemplo seria um cadastramento, onde é essencial que todo o cadastro seja enviado antes de direcionar as permissões.

 ### 2º Metodo 

 Inicia tudo de uma vez e vai finalizando conforme o tempo.

 Este método pode ser utilizado para o envio de emails, pois o email pode ser iniciado e, enquanto algo que precisa finalizar está sendo processado, o email é enviado.

 ### 3 Metodo

 Espera qualquer um dos processos terminar; o que terminar primeiro finaliza todo o processo.

Este método pode ser usado para verificação. O gateway de pagamento (a empresa que processar o pagamento primeiro, leva) utiliza esse método: o que finalizar primeiro, finaliza todo o processo.
